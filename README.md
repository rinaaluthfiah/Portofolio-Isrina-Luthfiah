## 📱 𝐃𝐞𝐬𝐤𝐫𝐢𝐩𝐬𝐢 𝐖𝐞𝐛𝐬𝐢𝐭𝐞

  Website portofolio ini dibuat sebagai tugas praktikum mata kuliah Pemrograman Web. Website ini dalam pembuatannya  menggunakan HTML, CSS, serta Bootstrap 5 untuk membantu proses layouting dan responsive design. Tujuan pembuatan website ini   adalah untuk menampilkan profil diri, kemampuan, pengalaman, serta sertifikat dalam bentuk tampilan yang rapi, modern, dan mudah diakses.
  
  Website terdiri dari tiga section utama (wajib), yaitu:
  
   ✦  Home
   
   ✦ About Me
   
   ✦ Certificates


## 🌟 𝐅𝐢𝐭𝐮𝐫 𝐖𝐚𝐣𝐢𝐛
  
  1️. Section Home (Hero Section) berisi perkenalan singkat
  
  2️. Section About Me berisi deskripsi diri, skills (progress bar), dan pengalaman
  
  
  3️. Section Certificates berisi daftar sertifikat dalam bentuk card atau layout grid
  
  
  4️. Minimal terdapat 1 Navbar
  
  
  5️. Minimal terdapat 1 Gambar

  6️. Menggunakan struktur dasar HTML yang benar (html, head, body)

  7️. Menggunakan CSS untuk styling (warna, background, layout, font, dll)


  8️. Tampilan tersusun rapi dan responsif.
