## 📱 𝐃𝐞𝐬𝐤𝐫𝐢𝐩𝐬𝐢 𝐖𝐞𝐛𝐬𝐢𝐭𝐞

  Website portofolio ini dibuat sebagai tugas praktikum mata kuliah Pemrograman Web. Website ini dalam pembuatannya  menggunakan HTML, CSS, serta Bootstrap 5 untuk membantu proses layouting dan responsive design. Tujuan pembuatan website ini   adalah untuk menampilkan profil diri, kemampuan, pengalaman, serta sertifikat dalam bentuk tampilan yang rapi, modern, dan mudah diakses.
  
  Website terdiri dari tiga bagian utama (wajib), yaitu:
  
   ✦  Home
   
   ✦ About Me
   
   ✦ Certificates


## 🌟 𝐅𝐢𝐭𝐮𝐫 𝐖𝐞𝐛𝐬𝐢𝐭𝐞 𝐏𝐨𝐫𝐭𝐨𝐟𝐨𝐥𝐢𝐨

1️⃣  Navbar – Menu navigasi untuk berpindah ke halaman Home, About Me, Certificates, dan Contact


2️⃣ Home Section – Menampilkan nama, bidang yang diminati, dan deskripsi singkat profil


3️⃣ About Me Section – Berisi latar belakang pendidikan, minat, pengalaman organisasi, serta pengembangan soft skill seperti        komunikasi dan kerja sama tim.


4️⃣ Certificates Section – Menampilkan sertifikat atau pencapaian sebagai bukti kompetensi


5️⃣ Contact Section – Menyediakan informasi kontak seperti email dan Instagram

 </> HTML – Digunakan untuk membuat struktur halaman seperti navbar, section Home, About Me, Certificates, dan Contact.

 🎨 CSS – Digunakan untuk mengatur tampilan website seperti warna, font, text-gradient, alignment, dan spacing.
